# Claude Sessions

This folder contains saved summaries from Claude chat sessions. Each note captures key outputs, decisions, code, and context from a conversation — used as a low-token reference layer for future chats.

## Naming Convention
`YYYY-MM-DD Topic-Keyword.md`

## How to Use
- At the end of a chat, paste the save prompt below and Claude will write a note here automatically
- To reload context in a future chat, say: *"Pull my Claude session note on [topic]"*
- Tag notes with relevant topics for searchability (e.g. `mersen`, `physics`, `formula-sae`)

## Save Prompt (copy-paste at end of any chat)
> Save this chat as a Claude session note in Obsidian under `Claude Sessions/`. Use today's date and a short topic keyword for the filename. Include: date, topic, key decisions or outputs, any important code or data, and open threads. Format it cleanly with frontmatter tags.
