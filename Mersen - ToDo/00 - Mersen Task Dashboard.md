# Mersen Task Dashboard

## How to Use

- **Quick task dump** → [[01 - Inbox Task Dump]] — throw anything here, no sorting required
- **Project-specific work** → [[02 - Active Projects]] → drill into the relevant note in `Projects/`
- **Blocked on someone else** → [[03 - Waiting On]] — log it and set a follow-up date
- **Mark complete** → change `- [ ]` to `- [x]` anywhere in the system
- **Worth archiving** → copy the completed item into [[04 - Completed Log]] with a date
